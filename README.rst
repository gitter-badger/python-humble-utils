Feature Areas
-------------

* File system:
    * file/directory paths extraction;
    * file/directory paths randomization;
    * file operations.
* String case convertions.
* Python class convenience shortcuts.
* `py.test`_ fixtures and helpers.

.. _py.test: https://docs.pytest.org/en/stable/
